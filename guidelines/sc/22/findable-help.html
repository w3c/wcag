Findable Help

A

New

For single page Web applications or any set of Web pages, if one or more of the following ways of finding help is supported, then access to at least one way of finding help is included in the same relative order on each page and page update:

  • Human contact details;
  • Human contact mechanism;
  • Self-help option;
  • A fully automated contact mechanism.

Access to help mechanisms may be provided directly on the page, or may be provided via a direct link to a different page containing the information"

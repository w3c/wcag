Animation from interactions

AAA

Proposed

This is a proposed Success Criterion that has not yet been formally approved by the Working Group. Discussion of the issue is available in Issue 18 and Pull Request 96. To file comments on this proposal, please raise new issues for each discrete comment in GitHub.

For non-essential motion or scaling animations triggered by a user action that is not an essential part of the action, there is a mechanism for the user to pause, stop or hide the animations while still performing the same action.

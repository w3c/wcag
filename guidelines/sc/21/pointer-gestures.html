Pointer Gestures

A

Proposed

This is a proposed Success Criterion that has not yet been formally approved by the Working Group. Discussion of the issue is available in Issue 61 and Pull Request 132. To file comments on this proposal, please raise new issues for each discrete comment in GitHub.

Functionality requiring complex or timed pointer gestures or multi-point gestures can also be operated with simple pointer gestures unless the use of multiple fingers and complex gestures is essential or content is operated in a multi-user environment.

Examples of a multi-pointer gesture include two-finger pinch/zoom

This requirement applies to web content which interprets pointer gestures (i.e. this does not apply to gestures that are required to operate the user agent or assistive technology)

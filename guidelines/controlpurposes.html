This section contains a listing of common user interface component purposes. The terms below are not keywords which must be used, but instead represent purposes that must be captured in the taxonomy adopted by a webpage. Where applicable, authors mark up controls with the chosen taxonomy to indicate the semantic purpose. This provides the potential for user agents and assistive technologies to apply personalized presentations that can enable more people to understand and use the content.

Common Control Purposes

The following items represent an intended destination or action.

  • Table of Contents - View or go to a table of contents
  • Next - View or go to the next item in a series (e.g. a page in a book or next article)
  • Previous - View or go to the previous item in a series
  • Reply - Reply to current item (e.g. an email)
  • Forward - Forward the current item (e.g. an email)
  • Inbox - View the inbox (e.g. an email application inbox)
  • Sent items - View items sent (e.g. an email application sent message folder)
  • Sign in - View or go to sign in or login page or process
  • Sign out - Sign out from current web site or application
  • Sign up - Register or sign up for a web site or application
  • Upload - Upload a file to a server
  • Refresh - Manually refresh data or refresh the contents of the screen
  • Print - Print the current page, selection, or item
  • Reset - Reset all controls on the page to their initial values
  • Comment - Submit a comment [AT RISK]
  • Settings - View and/or edit settings and options
  • Increase - Increase the value of the relevant setting
  • Decrease - Decrease the value of the relevant setting
  • Remove - Remove an item
  • Checkout - Proceed to the purchasing process
  • Open - Open an existing file or item
  • Compose - Create a new item (e.g. an email, online document, etc.)
  • Edit - Edit current item, selection, or the item at the current location
  • Cut - Remove the selected item or text and copy to the device clipboard
  • Copy - Copy the selected item or text to the device clipboard
  • Paste - Add the contents of the device clipboard in the current context
  • Save - Save the current content
  • Submit - Complete a process involving user input
  • Delete - Delete the relevant or current item
  • Undo - Revert to the state prior to the last change
  • Cancel - Close a dialog or similar container, discarding any unsaved changes
  • Home - View the initial or main page of a website or application
  • Contact us - View the contact information for content owner or producer [AT RISK]
  • About us - View information about the content owner or producer [AT RISK]
  • Our phone - Phone number [AT RISK]
  • Our email - Contact email [AT RISK]
  • Site map - View content containing a list of pages in a web site
  • Help - View the help function, support or instructions
  • Chat help - View a help function delivered via a chat interface
  • Services - View services available from the content provider [AT RISK]
  • Products - View products available from the content provider [AT RISK]
  • Terms - View terms and conditions [AT RISK]
  • Language - Manage language options
  • Social - View the content provider on social media [AT RISK]
  • My profile - View the user's profile page [AT RISK]

Common Input Control Purposes

The following input control purposes are intended to relate to the user of the content and pertain only to information related to that individual.

  • Name - Inputs used to handle information about a userâ€™s name(s) (e.g. first name, family name, suffix, etc.)
  • Professional Title - Job title (e.g., "Software Engineer", "Senior Vice President", "Deputy Managing Director")
  • Organization - Company name corresponding to the user
  • Address - Inputs used to handle address information (e.g. street address, city, region, postal code, etc.)
  • Country Code - A country's abbreviation code
  • Country Name - Full name of a country
  • Credit Card - Inputs used to handle information about credit card payments
  • Name on Credit Card - Full name as given on the payment instrument
  • Credit Card - Inputs used to handle information about credit card payments
  • Credit Card Number - Code identifying the payment instrument (e.g., the credit card number)
  • Credit Card Expiration Date - Expiration date of the payment instrument
  • Credit Card Security Code - Security code for the payment instrument (also known as the card security code (CSC), card validation code (CVC), card verification value (CVV), signature panel code (SPC), credit card ID (CCID), etc)
  • Credit Card Type - Type of payment instrument (e.g. Visa)
  • Transaction Currency - The monetary system being used for the transaction
  • Transaction Amount - The amount that the user is committing to for a transaction (e.g., when entering a bid or sale price)
  • Language - Preferred language
  • Birthdate - Birthday information
  • URL - Home page or other Web page corresponding to the user
  • Photo - Photograph, icon, or other image corresponding to the user
  • Telephone Number - Inputs used to handle information about telephone numbers
  • Email Address - Email address

Web Content Accessibility Guidelines 2

This resource contains editors' drafts of materials developed by the Accessibility Guidelines Working Group, using the w3c/wcag GitHub repository. The repository is currently used to maintain guidelines and support materials for WCAG 2.0, 2.1 and 2.2. Available editors' drafts:

  • Web Content Accessibility Guidelines 2.2 - snapshot of the WCAG 2.2 Recommendation with errata incorporated.
  • Web Content Accessibility Guidelines 2.1 - snapshot of the WCAG 2.1 Recommendation with errata incorporated.
  • Understanding WCAG 2 - editors' draft of Understanding WCAG 2, which supports WCAG 2.0 through 2.2.
  • WCAG 2 Techniques - editors' draft of WCAG 2 Techniques, which supports WCAG 2.0 through 2.2.
